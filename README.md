# BCtCI - Practice Directory

This directory is set up for running individual TypeScript files using **bun**.

## Running TypeScript Files

Run any `.ts` file directly:

```bash
bun run file.ts
```

## Installing Dependencies

If you need to install packages:

```bash
bun install <package-name>
```

## Example

Create a file `hello.ts`:

```typescript
console.log("Hello, world!");
```

Run it:

```bash
bun run hello.ts
```

## Bun Documentation

For more details: https://bun.sh/docs
